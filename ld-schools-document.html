01 · Overview

What is LD Schools?

LD Schools is a full-stack school management system built specifically for institutions that support students with Learning Disabilities — combining a complete ERP with an AI-driven LD intervention engine.

🏫

School ERP

Complete school management covering admissions, students, staff, fees, attendance, timetable, examinations, library, payroll, transport, and communications — all in one platform.

🧠

LD Platform

AI-powered screening, adaptive practice exercises, 5-level test system, Claude AI recommendations, parent scorecards, and teacher dashboards — built for dyslexia, dyscalculia & dysgraphia support.

03 · LD Platform

Learning Disability Support

An AI-powered intervention system uniquely built into the school platform — detecting, tracking, and supporting students with learning disabilities.

LD Types Supported

📖

Dyslexia

Reading, phonics, letter reversal difficulties

🔢

Dyscalculia

Number sense, math operations difficulties

✏️

Dysgraphia

Writing, fine motor, spelling difficulties

🔀

Mixed LD

Combination of multiple LD types

Student Journey

1. Phone OTP Login
Student registers via Supabase phone OTP
2. LD Screening (10 min)
Adaptive 10-question quiz across phonics, reading, writing, math
Claude AI
3. LD Classification
Risk score 0–100, LD type detected, personalised level assigned
4. Daily Practice
Adaptive exercises adjust difficulty based on performance (Elo rating)
5. Level Tests
5-level system — 70% pass rate required to unlock next level
6. AI Recommendations
Weekly personalised tips for student, teacher, and parent audiences
Claude AI
7. Re-screening every 90 days
Automatic re-screening prompt to track progress over time

07 · Setup Guide

Getting Started

Run the full platform locally in under 10 minutes using Docker, or install Node.js for development.

Option A — Docker (Recommended)

# 1. Copy environment file

cp .env.example .env

# 2. Fill in required values

nano .env

# 3. Start everything

docker compose up -d --build

# 4. Open in browser

open http://localhost

Option B — Local Dev (Node.js)

# Backend

cd backend

npm install && npm run dev

# Web frontend

cd web

npm install && npm run dev

# Open browser

http://localhost:5173
